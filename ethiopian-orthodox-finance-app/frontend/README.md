# Ethiopian Orthodox Tewahedo Church Finance Management Application - Frontend

This is the frontend part of the Ethiopian Orthodox Tewahedo Church Finance Management Application, built using React, Vite, and TypeScript. The application provides a user-friendly interface for managing financial activities, supporting bilingual functionality in French and Amharic.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/ethiopian-orthodox-finance-app.git
   cd ethiopian-orthodox-finance-app/frontend
   ```

2. Install dependencies:
   ```
   npm install
   ```

## Development

To start the development server, run:
```
npm run dev
```
This will start the application in development mode and open it in your default browser.

## Building for Production

To create a production build of the application, run:
```
npm run build
```
The production files will be generated in the `dist` directory.

## Localization

The application supports bilingual functionality. Localization files are located in the `public/locales` directory. You can add or modify translations in the `am.json` (Amharic) and `fr.json` (French) files.

## Testing

To run tests, use:
```
npm test
```
This will execute the test suite and provide feedback on the test results.
